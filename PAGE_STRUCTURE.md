# Monroe Urgent Care Page Structure

## Purpose

This document defines reusable page-level architecture for the new Monroe Urgent Care site. It intentionally avoids full page copy and focuses on section order, information hierarchy, conversion paths, and shared content needs.

## Global Page Rules

- Every core page should include a clear H1, a short orientation statement, and at least one conversion path.
- Patient pages should prioritize Book, Call, and Directions.
- Employer pages should prioritize Request Employer Services, Call, and Directions.
- Emergency-care guidance should appear wherever a visitor may be deciding between urgent care and the ER.
- Hours, address, and phone should be easy to access from any page.
- IV hydration should be treated as a secondary wellness path, not the dominant urgent-care conversion.
